# User List - Front-end

## Použité technologie
- React
    - react-bootstrap
    - react-redux
- Redux
    - redux-thunk
    - redux-form
- axios

## Instalace a spuštění

1. Ujistěte se, že máte spuštěný server v nadřazené složce 

2. Nainstalujte závislosti
```
$ npm install
```

3. Nastartujte React dev server

```
$ npm start
```
